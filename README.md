# Slingshot

**Slingshot** is a turn-based 2-player shooter set in space.  
The objective is to destroy the other player ship by shooting projectiles  
through space, using the planets gravity fields as guidance.

The game is based on a classic open-source game developed by Jonathan Musther.

> ## The project
>
> The project uses the **Unity** engine.  
> All the physics and controls are written from scratch, in C# language.
